# Academic GitHub Homepage

This is a minimal black-and-white academic homepage designed for GitHub Pages.

## How to publish

1. Create or reuse the repository `TMSDMAP.github.io`.
2. Put `index.html` and `style.css` in the repository root.
3. Commit and push to GitHub.
4. Open `https://TMSDMAP.github.io`.

If your actual GitHub username is different, the repository must be named
`YOUR_USERNAME.github.io`.

## What to replace

- `assets/profile.jpg` if you want to update the portrait
- news items
- selected projects as your GitHub portfolio changes
- Google Scholar link after the profile is available

## Suggested repository structure

```text
TMSDMAP.github.io/
  index.html
  style.css
  个人简历.pdf
  assets/
    profile.jpg
```

The current portrait is stored at `assets/profile.jpg` and displayed as a
small circular image.
